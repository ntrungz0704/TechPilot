# CP-03.0 — Cleanup redundant V1 codebase files from repository root

Checkpoint này thực hiện dọn dẹp (xóa) các thư mục và file của mã nguồn V1 cũ tại root của repository để làm sạch cấu trúc dự án.

## Objective

- Xóa các thư mục cũ dư thừa ở root: `app`, `config`, `database`, `public`.
- Xóa file entry point cũ: `index.php`.

## Scope

- Cho phép xóa các thư mục và file được chỉ định ngoài `techpilot/`.
- Không được thay đổi bất kỳ file nào trong `techpilot/`.
