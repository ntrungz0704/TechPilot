# CP-05.0 — Align codebase to MVC standards, fix assets loading, and clean up redundant files

Checkpoint này thực hiện khắc phục lỗi mất CSS/JS do đường dẫn tĩnh web server, sửa tên JS core và dọn dẹp các tệp tin dư thừa không sử dụng trong dự án.

## Objective

- Cấu hình lại `router.php` ở root để phục vụ chính xác các file tĩnh trong `public/assets/` khi server chạy từ root.
- Đồng bộ tên file JS chính thành `main.js` khớp với views.
- Xóa bỏ các file CSS và hình ảnh dư thừa (như `admin.css`, `Untitled.png`).
