# CP-04.0 — Move V2 codebase files to repository root

Checkpoint này thực hiện di chuyển toàn bộ mã nguồn phiên bản V2 từ thư mục con `techpilot/` ra ngoài thư mục root để tối ưu hóa cấu trúc dự án.

## Objective

- Di chuyển toàn bộ các thư mục và file của V2 ra root: `app`, `config`, `database`, `design`, `public`.
- Sửa đổi cấu hình đường dẫn `ROOT_PATH` và `BASE_URL` cho tương thích với root mới.
- Khởi chạy lại PHP server phục vụ từ root.
