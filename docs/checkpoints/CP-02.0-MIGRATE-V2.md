# CP-02.0 — TechPilot V2 Migration and ERD 2 Pages

Checkpoint này thực hiện di chuyển cấu trúc vận hành của ứng dụng sang thư mục `/techpilot` (sử dụng cấu trúc Database V2) và hiện thực hóa các trang chức năng ERD 2 còn thiếu.

## Objective

- Đưa ứng dụng V2 trong `techpilot/` vào hoạt động hoàn chỉnh trên môi trường local.
- Viết mới các trang: Yêu thích (Wishlist), So sánh sản phẩm (Compare), Lịch sử đơn hàng, Đổi trả sản phẩm và Thông báo hệ thống.

## Scope

- Tạo file cấu hình `techpilot/config/app.php` và `techpilot/config/database.local.php`.
- Tạo mới các file controller, model và view cho các tính năng mới trong `techpilot/app/`.
- Cập nhật header/footer của layout `techpilot/app/views/layouts/`.
- Import schema V2 vào cơ sở dữ liệu local.
