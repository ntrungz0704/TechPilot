# CP-01.0 — UI Enhancement: Banners and Partner Logos

Checkpoint này thực hiện cải tiến trực quan cho trang chủ của TechPilot bằng cách nạp các ảnh banner chất lượng cao và sửa lỗi hiển thị logo đối tác chiến lược.

## Objective

- Cung cấp ảnh nền chất lượng cao cho banner "Trả góp 0% Lãi Suất", "Thu Cũ Đổi Mới" và "Trả Góp Qua Thẻ Tín Dụng".
- Khắc phục lỗi hiển thị text thay vì logo thật của đối tác chiến lược bằng cách cập nhật đuôi ảnh logo sang SVG trong cơ sở dữ liệu.

## Scope

- Tạo mới 3 ảnh banner tĩnh trong thư mục `public/assets/images/`.
- Tinh chỉnh CSS trong `public/assets/css/style.css`.
- Chạy truy vấn SQL cập nhật dữ liệu database local.

## Out of scope

- Mọi thay đổi trong mã nguồn PHP của `techpilot/**`.
- Thay đổi cấu trúc cơ sở dữ liệu (schema migrations).
